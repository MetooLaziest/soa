## 目标（Objective）

修复产品发布页（productsfuture.html）及其全部子页面（product-P001~P022）的菜单栏样式异常问题，使全站导航栏显示一致。

---

## 关键推理（Key Reasoning）

### 问题根因
通过对比首页（index.html）和产品发布页（productsfuture.html）的 HTML 源码，发现：

1. **首页正确引用**：
   ```html
   <link href="https://...bootstrap.min.css" ...>
   <link rel="stylesheet" href="css/global.css">
   ```

2. **产品发布页错误引用**：
   ```html
   <link rel="stylesheet" href="css/style.css">          ← 罪魁祸首
   <link href="https://...bootstrap.min.css" ...>
   <link rel="stylesheet" href="css/global.css">
   ```

3. **style.css 的问题**：
   - 这是 Blocs 旧模板遗留的 CSS 文件
   - 内含大量 `!important` 声明的导航栏样式
   - 例如：`.navbar { background-color: rgba(0,0,0,0.7) !important; }`
   - 这些 `!important` 规则覆盖了 `global.css` 的正确样式

### 解决方案
删除所有页面中的 `<link rel="stylesheet" href="css/style.css">` 引用，让导航栏样式完全由 `global.css` 控制。
