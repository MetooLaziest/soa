# 机伴管理 - 完整 12 字段表单

## 任务
扩展机伴管理表单，从基础字段升级为完整 12 字段配置

## 数据库字段（companion_models）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| name | text | 型号名称 |
| model_code | text | 型号代码 |
| description | text | 人格描述 |
| avatar_asset_id | uuid | 关键立绘 |
| catchphrases | text[] | 口头禅 |
| system_prompt_template | text | System Prompt 模板 |
| contractor_setting | text | 契约者设定 |
| initial_node_id | uuid | 初始化剧情节点 |
| voice_id | text | 语音播报音色 |
| mbti_base | jsonb | MBTI 四维分数 |
| knowledge_scope | text | 知识范围 |
| initial_items | text | 初始道具 |

## 新增字段
- `knowledge_scope` TEXT - 知识范围（逗号分隔数组）
- `initial_items` TEXT - 初始道具（逗号分隔数组）

## 表单字段映射

| 表单字段 | 数据库字段 | 控件类型 |
|----------|-----------|----------|
| 型号名称 | name | 文本输入 *必填 |
| 型号代码 | model_code | 文本输入 *必填 |
| 人格描述 | description | 多行文本 |
| 契约者设定 | contractor_setting | 多行文本 |
| 初始化剧情节点 | initial_node_id | 文本输入(UUID) |
| 语音播报音色 | voice_id | 文本输入 |
| 关键立绘(资源ID) | avatar_asset_id | 文本输入(UUID) |
| 知识范围 | knowledge_scope | 文本+快捷按钮 |
| 初始属性(MBTI) | mbti_base | 4个 range slider |
| 口头禅 | catchphrases | 文本输入(逗号分隔) |
| System Prompt模板 | system_prompt_template | 多行文本(Mono字体) |
| 初始道具 | initial_items | 文本输入(逗号分隔) |
