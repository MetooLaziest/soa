## E-Pet Bug - statusPill 空指针导致藏品库崩溃

### 时间
- 2026-06-02 10:08 UTC+8

### 问题
用户点击藏品库选择展示小宠物时报 `TypeError: Cannot set properties of null (setting 'className')`，藏品库无法使用。

### 根因
`updateUI()` 函数第 558 行执行 `document.getElementById('statusPill')` 返回 `null`，因为 HTML 中根本不存在 `id="statusPill"` 的元素。v7 UI 重构时移除了状态栏 DOM 节点，但 `updateUI()` 里的引用未被清理。

同时 `updateUI()` 直接引用全局 `state` 变量（未定义），而应该从 `visiblePets[0]._state` 取值。

### 修复
1. **HTML** — 在 `.container` 内添加 `<div id="statusPill" class="status-pill"></div>`
2. **updateUI()** — 改为遍历 `visiblePets` 计算每只宠物的 `._state`，用第一只宠物的状态更新 `statusPill`
3. **loadVisiblePets()** — 每次加载宠物后同步计算 `._state`

### 修复文件
- 临时文件: `/tmp/epet_live.html`

**原因**：`eatingPetId` 是普通 JS 变量，`renderPetCards` 每 15 秒重建 DOM 时会重置为 null

**解决**：`eatingPetId` 改存 sessionStorage，每次 `renderPetCards` 开头恢复：
```javascript
// renderPetCards 开头
eatingPetId = sessionStorage.getItem('eatingPetId') || null;

// doFeed 里持久化
sessionStorage.setItem('eatingPetId', petId);
// 5秒后清除
setTimeout(() => {
  sessionStorage.removeItem('eatingPetId');
  eatingPetId = null;
}, 5000);
```

```javascript
const lastPetted = lastPettedAt[petId] || 0;

```
每次 15s 刷新后，只要在 600ms 窗口内，抖动状态依然保留。

## E-Pet 两个问题修复（桌子误触 + 原地抖动）

### 时间
- 2026-06-02 13:29 CST

### 问题1: 桌子触摸区误触
**原因**：点击 pet-orb/img 后事件仍会冒泡到 table-zone（pet-orb → pet-area → container → table-zone）

**解决**：建立完整的 pointer-events 链路
```css
.pet-area { pointer-events: none; }      /* 中间层不拦截 */
.pet-area .pet-img { pointer-events: auto; }  /* 图片接收点击 */
.pet-orb { pointer-events: none; }      /* 外层穿透 */
```
点击 img → `img.onclick` → `e.stopPropagation()` → 不冒泡到 table-zone

### 问题2: 抖动时跑回原位
**原因**：`shakeAnim` 用 `translateX` 会和 `tableWalk` 的 transform 冲突，导致宠物脱离椭圆轨道

**解决**：`shakeAnim` 只用 `rotate`，不用 `translateX`

## E-Pet 旧版藏品库按钮点击无效修复

### 时间
- 2026-06-02 17:32 CST

### 问题
旧版（`/epet/`）右上角藏品库按钮点击无效。

### 根因
`.collection-btn` 在 `.pet-area` 内部，而 `.pet-area` 为防误触桌子设置了 `pointer-events: none`，`.collection-btn` 没有显式覆盖，导致点击事件被吞掉。

### 修复
```css
.collection-btn {
  pointer-events: auto; /* 显式恢复可点击 */
  position: fixed;
  top: 16px;
  right: 16px;
  z-index: 100;
  background: rgba(255,255,255,0.15);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255,255,255,0.3);
  border-radius: 50%;
  width: 44px;
  height: 44px;
  font-size: 20px;
  cursor: pointer;
}
```
