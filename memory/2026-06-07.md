### 问题根因确认
用户确认看到 **RAGEdit 加载中... [isNew=, id=]**

这说明：
- 路由正确匹配到了 **RAGEdit 组件**（不是 RAGList）✅
- 但 `id` 参数是**空值**，导致 `isNew=false`，所以 `loading=true` 卡住 ❌

**根因**：`/rag/new` 路由在 App.tsx 中定义时没有 `:id` 参数，所以 `useParams()` 从该路由取到的 `id` 是空字符串/ undefined。

### 修复方案
改用 `location.pathname.endsWith('/rag/new')` 判断是否新建模式，不再依赖 `id === 'new'`：

```tsx
// 修复前
const { id } = useParams<{ id: string }>();
const isNew = id === 'new';

// 修复后
const location = useLocation();
const isNew = location.pathname.endsWith('/rag/new');
```
