# 任务总结 - soa.laziestlife.com 前端无法访问修复

**时间**: 2026-05-16 09:09-09:11
**问题**: https://soa.laziestlife.com 返回 500 错误

## 根因
后端 `index.js` 配置静态文件路径为 `__dirname + '/../frontend'`，即 `/var/www/iot-ai-doll/backend/frontend/`，但该目录不存在。前端构建产物实际位于 `/var/www/iot-ai-doll/frontend/dist/`。

错误信息：`ENOENT: no such file or directory, stat '/var/www/iot-ai-doll/backend/frontend/index.html'`

## 修复
创建软链接：`ln -sfn /var/www/iot-ai-doll/frontend/dist /var/www/iot-ai-doll/backend/frontend`

## 验证
- `curl https://soa.laziestlife.com/` → HTTP 200 ✅，返回前端 HTML ✅
