### 4. 功能支持
- WebSocket长连接 + 心跳保活
- ASR: Tencent/火山引擎/讯飞 自动降级
- TTS: Tencent/火山引擎/讯飞 自动降级
- LLM: Kimi (优先) / OpenAI (备选)
- 会话管理 + Redis持久化(可选)
- Debug模式(无APIKey时测试)

### 5. ESP32设备协议
定义了完整的JSON通信协议：
- register / registered
- audio / asr / llm / audio(TTS)
- reset
