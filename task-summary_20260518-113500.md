## 执行时间

## 问题诊断

### 1. UI 样式丢失
- **原因**：Tailwind CSS v4 需要在 CSS 文件中添加 `@import "tailwindcss"` 指令
- **修复**：在 `src/index.css` 头部添加 Tailwind 导入和自定义主题配置

### 2. 核心功能缺失
根据参考系统，缺少以下管理功能：
- AI 模型配置
- 系统配置管理
- 用户列表
- 设备 SN 录入
- AI 资源与计费
- AI 对话风格设定

## 解决方案

### 修复 Tailwind CSS
```css
/* src/index.css */
@import "tailwindcss";

@theme {
  --color-dark: #0f0f13;
  --color-card: #1a1b23;
  --color-purple-500: #8b5cf6;
  --animate-fade-in-up: fade-in-up 0.3s ease-out;
}
```

### 新增 6 个管理页面

| 页面 | 路由 | 核心功能 |
|------|------|---------|
| AI 模型配置 | `/admin/ai-models` | 编辑模型参数、API endpoint、temperature 等 |
| 系统配置 | `/admin/sys-config` | 键值对配置管理，支持增删改 |
| 用户列表 | `/admin/users` | 用户管理 + 设备绑定查看 |
| 设备 SN | `/admin/devices` | 单个/批量录入设备序列号 |
| AI 资源 | `/admin/ai-resources` | Token 用量统计、费用统计、模型分布 |
| 对话风格 | `/admin/ai-personality` | 对话风格和性格特质管理 |
